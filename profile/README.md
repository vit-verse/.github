# VIT Verse

Student driven tools and utilities built for VIT students.

---

## ⚠️ Project Status

**VIT Verse has been discontinued effective April 15, 11:59 PM.**

This decision follows official instructions from VIT/SDC regarding third party applications accessing the VTOP portal.

---

## 📌 Overview

VIT Verse was developed as a simple and creative solution to help students with:

- Academic tracking and insights  
- Attendance and timetable utilities  
- Campus related tools and features  
- Centralized access to student services  

The goal was to provide a streamlined and efficient experience for managing student activities.
